Agent-based taxi simulator to test strategies


* Free software: MIT license
* Documentation: https://taxi-simulator.readthedocs.io.


Features
--------

* Strategy pattern
* Continuous simulator
* Load scenarios
* Multi-agent system built with SPADE_
* XMPP communications
